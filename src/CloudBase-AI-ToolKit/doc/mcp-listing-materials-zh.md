## 安装

### MCP 配置
```json
{
  "mcpServers": {
    "cloudbase-mcp": {
      "command": "npx",
      "args": ["npm-global-exec@latest", "@cloudbase/cloudbase-mcp@latest"]
    }
  }
}
```

## 核心功能

- 微信小程序开发和部署
- 无服务器云函数和数据库
- 一键部署到生产环境
- 微信生态系统的中国市场准入
- 内置云开发知识库
- 34个专业MCP工具
